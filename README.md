# Movies app 🎉

Movie app for search movies from OMDb.

## 💻 Techs

- Next.js
- OMDb public API

## 📦 Installation

### 📰 Clone the project

```bash
git clone https://github.com/emreunsal1/movies-case.git
```

### 🔻 Install Dependencies

```bash
npm install
```

### 🏃‍♂️ Start the project

```
npm run build && npm run start
```
